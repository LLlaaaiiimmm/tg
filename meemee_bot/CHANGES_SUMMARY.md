## Что нужно для запуска:

### 1. Проверить .env файл
Убедитесь, что в `/app/meemee_bot/.env` есть:
- `BOT_TOKEN` - токен бота от @BotFather
- `REDIS_URL` - URL Redis (например, `redis://localhost:6379`)
- `KIE_AI_API_KEY` - API ключ для генерации видео

### 2. Запустить Redis
```bash
# Если Redis не установлен:
sudo apt-get install redis-server
sudo systemctl start redis

# Или через Docker:
docker run -d -p 6379:6379 redis
```

### 3. Установить зависимости (если не установлены)
```bash
cd /app/meemee_bot
npm install
```

### 4. Запустить бота
```bash
cd /app/meemee_bot
npm start
```
