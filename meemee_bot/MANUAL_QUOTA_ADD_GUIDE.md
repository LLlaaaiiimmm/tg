# Инструкция по ручному добавлению видео клиенту

## Проблема
Клиент оплатил 50 видео через крипту, платёж прошёл, но видео не добавились в аккаунт.

**Детали платежа:**
- Payment ID: 12369503
- Billing ID: CRYPTO-20251103-9730890680
- Amount: 2.5 USDT (BEP20)
- Client ID: 1916527652
- Date: 11/03/2025 14:07
- Должен получить: 50 видео

## Исправленные баги

### 1. ❌ Критический баг в webhook handler
**Проблема:** В `/app/meemee_bot/src/backend/index.js` использовался `clientId` из webhook вместо `order.userId`

**Исправление:**
```javascript
// БЫЛО:
await userService.addPaidQuota(clientId, pkg.generations);

// СТАЛО:
await userService.addPaidQuota(order.userId, pkg.generations);
```

### 2. ✅ Добавлено улучшенное логирование
Теперь webhook выводит полную информацию о полученных данных для диагностики.

### 3. ✅ Добавлена проверка пакета
Теперь webhook проверяет, что пакет существует перед добавлением генераций.

## Как добавить видео вручную клиенту

### Шаг 1: Убедитесь что Redis запущен
```bash
service redis-server start
redis-cli ping
# Должен вывести: PONG
```

### Шаг 2: Запустите скрипт добавления квоты
```bash
cd /app/meemee_bot
node add_quota_manually.js <USER_ID> <КОЛИЧЕСТВО_ВИДЕО>
```

**Для данного клиента:**
```bash
cd /app/meemee_bot
node add_quota_manually.js 1916527652 50
```

### Ожидаемый результат:
```

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
👤 User ID: 1916527652
📊 Old paid quota: 0
➕ Added: 50
📊 New paid quota: 50
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### Шаг 3: Проверьте результат
```bash
redis-cli GET "user:1916527652"
```

Вы должны увидеть JSON с `"paid_quota": 50`

## Важные заметки

### Конфигурация пакетов
Проверьте `/app/meemee_bot/src/config.js` - там должен быть правильный пакет за 2.5 USDT.

**Текущие пакеты:**
- single: 1 видео = 5.8 USDT
- pack_5: 5 видео = 29 USDT
- pack_10: 10 видео = 58 USDT
- pack_50: 50 видео = 290 USDT

⚠️ **ВНИМАНИЕ:** Если клиент заплатил 2.5 USDT за 50 видео, возможно:
1. Был специальный промокод/скидка
2. Неправильная конфигурация цен
3. Тестовый платёж

В любом случае, скрипт `add_quota_manually.js` позволяет добавить любое количество видео вручную.
