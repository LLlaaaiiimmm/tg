## 🚀 Как использовать

### Запуск админ-бота

1. Настройте `.env` файл (скопируйте из `.env.example`):
```bash
cd /app/meemee_bot
cp .env.example .env
```

2. Убедитесь, что Redis запущен:
```bash
redis-server --daemonize yes
```

3. Установите зависимости (если нужно):
```bash
yarn install
```

4. Запустите админ-бота:
```bash
node src/bot_start_admin.js
```

### Использование функционала

1. Откройте админ-бота в Telegram
2. Отправьте `/start`
3. Нажмите `👥 Пользователи`
4. Отправьте ID пользователя (число)
5. Используйте кнопки для управления генерациями

## 📋 Примеры операций

## 🔧 Технические детали

### Используемые методы UserService:

- `getUser(userId)` - получение пользователя
- `addFreeQuota(userId, amount)` - добавление бесплатных генераций
- `removeFreeQuota(userId, amount)` - удаление бесплатных генераций
- `addPaidQuota(userId, amount)` - добавление платных генераций

### Обработчики callback:

- `add_quota_(\d+)` - меню добавления генераций
- `add_quota_confirm_(\d+)_(\d+)` - подтверждение добавления
- `add_quota_custom_(\d+)` - ручной ввод для добавления
- `remove_quota_(\d+)` - меню удаления генераций
- `remove_quota_confirm_(\d+)_(\d+)` - подтверждение удаления
- `remove_quota_custom_(\d+)` - ручной ввод для удаления
- `show_user_(\d+)` - просмотр пользователя
