### 2. Реализованы динамические плейсхолдеры

**Поддерживаемые плейсхолдеры:**
- `{name}` - Имя ребенка (Ростик, Маша, и т.д.)
- `{gender_text}` - Пол на русском (мальчик/девочка)
- `{gender_child}` - Пол на английском (boy/girl)
- `{gender_pronoun}` - Местоимение (He/She)
- `{gender_possessive}` - Притяжательное (his/her)
- `{gender_object}` - Объектное (him/her)
- `{gender_full_description}` - Полное описание (полный мальчик славянской национальности / полная девочка славянской национальности)

### 3. Обновлен GenerationService

**Файл:** `/app/meemee_bot/src/services/Generation.service.js`

**Новые методы:**

#### `getGenderReplacements(gender)`
Возвращает все варианты замены для указанного пола:
```javascript
// Для male:
{
  gender_text: 'мальчик',
  gender_child: 'boy',
  gender_pronoun: 'He',
  gender_possessive: 'his',
  gender_object: 'him',
  gender_full_description: 'полный мальчик славянской национальности'
}

// Для female:
{
  gender_text: 'девочка',
  gender_child: 'girl',
  gender_pronoun: 'She',
  gender_possessive: 'her',
  gender_object: 'her',
  gender_full_description: 'полная девочка славянской национальности'
}
```

#### `replacePlaceholders(obj, replacements)`
Рекурсивно заменяет все плейсхолдеры в объекте/строке:
- Поддерживает строки
- Поддерживает массивы
- Поддерживает вложенные объекты
- Использует регулярные выражения для глобальной замены

### 4. Обработка разных типов промптов

Теперь `createGeneration()` поддерживает:
1. **Простые строковые промпты** (как раньше)
   ```json
   {
     "prompt": "Создай видео где {gender_text} {name}..."
   }
   ```

2. **Сложные JSON промпты** (новое)
   ```json
   {
     "prompt": {
       "scene": "...",
       "visual": "...",
       "audio": {...}
     }
   }
   ```

### 5. Обновлен метод generateVideo
