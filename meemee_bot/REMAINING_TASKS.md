## ❌ ЧТО НУЖНО ДОДЕЛАТЬ (5%)

### 1. API Ключи (КРИТИЧНО)

**Файл:** `/app/meemee_bot/.env`

Нужно получить и добавить следующие ключи:

```bash
# 1. Telegram Bot Token
BOT_TOKEN=ваш_токен              # От @BotFather
BOT_TOKEN_ADMIN=ваш_токен_админа # От @BotFather (второй бот)

# 2. Kie.ai API (для генерации видео)
KIE_AI_API_KEY=ваш_kie_ai_ключ   # От kie.ai

# 3. Lava Top (оплата картами)
LAVA_PAYMENT_API=ваш_lava_ключ   # От lava.top

# 4. 0xprocessing (оплата криптой)
PAYMENT_API=ваш_0xprocessing_ключ     # От 0xprocessing.com
MERCHANT_ID=0xMR1234567               # Ваш Merchant ID

# 5. Поддержка
SUPPORT_USERNAME=ваш_telegram         # Ваш Telegram username
```

**Где получить:** См. файл `API_SETUP_GUIDE.md`

---

### 2. Lava Offer IDs (КРИТИЧНО)

**Файл:** `/app/meemee_bot/src/config.js`

Нужно создать 4 Offer в личном кабинете Lava и добавить их ID:

```javascript
// Строка 12
offerIdLava: 'YOUR_LAVA_OFFER_ID_SINGLE'  ❌ → заменить на реальный

// Строка 22
offerIdLava: 'YOUR_LAVA_OFFER_ID_PACK10'  ❌ → заменить на реальный

// Строка 32
offerIdLava: 'YOUR_LAVA_OFFER_ID_PACK100' ❌ → заменить на реальный

// Строка 42
offerIdLava: 'YOUR_LAVA_OFFER_ID_PACK300' ❌ → заменить на реальный
```

**Цены для Offer:**
- 1 видео = 580₽
- 10 видео = 5000₽
- 100 видео = 40000₽
- 300 видео = 100000₽

### 4. Webhook URLs (ДЛЯ ПРОДАКШЕНА)

После деплоя на сервер с доменом, настроить в панелях:

**Lava Top:**
- Webhook URL: `https://ваш-домен.com/webhook/lava`
- Метод: POST
- Секрет: значение из `WEBHOOK_PASSWORD` в .env

**0xprocessing:**
- Webhook URL: `https://ваш-домен.com/webhook/crypto`
- Метод: POST
- Секрет: значение из `WEBHOOK_PASSWORD_PROCESSING` в .env
