# Реализация кнопки "Использовать бесплатную генерацию"

## Описание изменений

Добавлена динамическая кнопка в главное меню бота, которая позволяет пользователям быстро использовать свои бесплатные генерации.

## Реализованная функциональность

### 1. Динамическая клавиатура главного меню
- **Файл**: `src/screens/keyboards.js`
- **Функция**: `createMainMenuKeyboard(userId)`
- **Логика**:
  - Получает данные пользователя из Redis
  - Проверяет наличие бесплатных генераций (`free_quota > 0`)
  - Если есть бесплатные генерации, добавляет кнопку в начало меню
  - Возвращает персонализированную клавиатуру

### 2. Кнопка бесплатной генерации
- **Текст**: `🎁 Использовать бесплатную генерацию (осталось: X)`
- **Callback data**: `use_free_generation`
- **Условие показа**: Показывается ТОЛЬКО когда `free_quota > 0`
- **Позиция**: Первая кнопка в главном меню (если доступна)

### 3. Обработчик кнопки
- **Файл**: `src/bot_start.js`
- **Action**: `use_free_generation`
- **Логика**:
  1. Проверяет наличие бесплатных генераций у пользователя
  2. Если генераций нет, показывает уведомление
  3. Если генерации есть, перенаправляет на каталог мемов
  4. Пользователь может выбрать мем и сгенерировать видео

### 4. Интеграция в бот
- Обновлены все точки входа в главное меню:
  - `/start` команда
  - Callback `main_menu`
  - После регистрации
  - После реферальной регистрации

## Сценарии использования

### Сценарий 1: Пользователь использует бесплатную генерацию
1. Пользователь открывает бота
2. Видит кнопку "🎁 Использовать бесплатную генерацию (осталось: 1)"
3. Нажимает на кнопку
4. Бот показывает каталог мемов
5. Пользователь выбирает мем и создает видео
6. После использования, кнопка исчезает из главного меню

### Сценарий 2: Пользователь без бесплатных генераций
1. Пользователь открывает бота
2. НЕ видит кнопку бесплатной генерации
3. Может купить видео через кнопку "💳 Купить видео"

### Сценарий 3: Попытка использовать несуществующую генерацию
1. Пользователь нажимает кнопку (edge case)
2. Бот проверяет наличие генераций
3. Если генераций нет, показывает уведомление: "❌ У вас нет бесплатных генераций"
4. Пользователь не может продолжить

## Технические детали

### Асинхронность
- Функция `createMainMenuKeyboard()` является асинхронной
- Требует использования `await` при вызове
- Гарантирует актуальные данные из Redis

### Проверки безопасности
- Двойная проверка наличия квоты (в меню и при нажатии)
- Защита от race conditions
- Валидация пользователя перед показом кнопки

### Производительность
- Минимальная задержка (один запрос к Redis)
- Кеширование не требуется, т.к. данные всегда актуальны
